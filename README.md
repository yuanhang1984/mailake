# 目录结构(2018.04.25更新)
--[doc] 产品文档目录
----[产品设计] 产品设计的相关文件

--[lib] js库
----[environment] 环境相关
----[toolkit] 工具相关
--[module] 模块目录
----[mod_name] 模块名称目录
------[default_theme] 模块主题目录
--------[css] 主题下css目录
--------[img] 主题下img目录
--[plugin] 插件目录
----[jquery] JQuery插件
------[3.3.1] 版本号
--------jquery.min.js 文件
--favicon.ico 显示图标
--README.md 此文件
